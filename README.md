# Bubble
# Классификация концентрации пузырьков в растворах

Автоматическая классификация 7 уровней концентрации (0-100%) на основе видеокадров с использованием transfer learning (ResNet50). Проект создан для анализа supplementary materials к научной статье ACS.

## Результаты

**Test Accuracy**: 94.8%  
**Test F1-Score**: 0.947  
**Test Precision**: 0.949  
**Test Recall**: 0.948

### По классам:

| Класс | Precision | Recall | F1-Score | Support |
|-------|-----------|--------|----------|---------|
| **0%** | 0.958 | 1.000 | 0.979 | 23 |
| **5%** | 0.989 | 1.000 | 0.994 | 86 |
| **12.5%** | 1.000 | 1.000 | 1.000 | 77 |
| **25%** | 0.977 | 0.977 | 0.977 | 86 |
| **50%** | 0.891 | 0.983 | 0.934 | 58 |
| **75%** | 0.926 | 0.808 | 0.863 | 78 |
| **100%** | 0.857 | 0.889 | 0.873 | 54 |

**Вывод**: Низкие концентрации (0-25%) распознаются почти идеально. Высокие (50-100%) путаются между собой из-за визуальной схожести.

## Датасет

**Источник**: Supplementary materials к статье [jp2c00948](https://pubs.acs.org/doi/10.1021/jp2c00948) (ACS Publications)

**Структура**:
- 7 видеофайлов (.mp4), по одному на каждую концентрацию
- Из видео извлекаются кадры с фильтрацией по информативности
- Итоговый датасет: ~460-500 изображений после всех фильтров

**Классы**:
- `solution_0pct` — чистый раствор
- `solution_5pct` — 5% концентрация
- `solution_12.5pct` — 12.5% концентрация
- `solution_25pct` — 25% концентрация
- `solution_50pct` — 50% концентрация
- `solution_75pct` — 75% концентрация
- `solution_100pct` — 100% концентрация
