# Классификация концентрации пузырьков в растворах

Автоматическая классификация 7 уровней концентрации (0-100%) на основе видеокадров с использованием transfer learning (ResNet50). Проект создан для анализа supplementary materials к научной статье ACS.

## Результаты

**Test Accuracy**: 94.8%  
**Test F1-Score**: 0.947  
**Test Precision**: 0.949  
**Test Recall**: 0.948

## Датасет

**Источник**: Supplementary materials к статье [jp2c00948](https://pubs.acs.org/doi/10.1021/jp2c00948) (ACS Publications)

**Структура**:
- 7 видеофайлов (.mp4), по одному на каждую концентрацию
- Из видео извлекаются кадры с фильтрацией по информативности
- Итоговый датасет: ~460-500 изображений после всех фильтров

**Классы**:
- `solution_0pct` — чистый раствор
- `solution_5pct` — 5% концентрация
- `solution_12.5pct` — 12.5% концентрация
- `solution_25pct` — 25% концентрация
- `solution_50pct` — 50% концентрация
- `solution_75pct` — 75% концентрация
- `solution_100pct` — 100% концентрация

1.py — извлечение кадров

Скрипт парсит исходные видео, вытаскивает фреймы с фиксированным интервалом и сразу фильтрует тёмные/пустые кадры по порогу яркости. На выходе получаем сырые изображения, разложенные по папкам концентраций.
2.py — приведение к единому размеру

Все изображения ресайзятся до 224×224. Пропорции сохраняются, добавляется чёрный.
2.5.py — фильтрация по содержанию (пузырьки)

Здесь уже смотрим не на яркость, а на наличие целевого объекта. Кадры без пузырьков или с артефактами отбрасываются. 
3.py - сама модель

Загружает отфильтрованные кадры, делает  сплит 70/15/15. На train применяются аугментации (флип, поворот, цветовой джиттер), на val/test только нормализация ImageNet. Берётся предобученный ResNet50, бэкбон полностью замораживается, голова заменяется на Dropout(0.5) → Linear(2048→256) → ReLU → Dropout(0.3) → Linear(256→7). Обучение идёт 50 эпох с Adam и StepLR. Сохраняется чекпоинт с лучшим val accuracy и минимальным разрывом train/val. После обучения модель тестируется, строятся графики.
Анализ результатов
Модель сходится быстро, к 10-й эпохе выходит на плато. Val accuracy стабильно выше train.
Confusion Matrix: низкие концентрации (0, 5, 12.5, 25%) распознаются почти идеально. Высокие значения (50, 75, 100%) модель путает  между собой: 75% часто уходит в 50% или 100%, 100% частично смешивается с 75%.
